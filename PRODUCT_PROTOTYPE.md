# LureBin 路亚小程序 - 功能原型

## 产品定位
路亚爱好者的一站式平台：记录 + 学习 + 交易

---

## 信息架构

```
LureBin
├── 首页（推荐/动态）
├── 钓点（地图/列表）
├── 发布（+ 按钮）
├── 市场（二手交易）
└── 我的（个人中心）
```

---

## 页面详情

### 1. 首页 🏠

**功能：**
- 顶部：搜索栏（搜钓点、拟饵、商品、资讯）
- Banner：推荐内容/活动
- **Tab 切换**：关注 | 推荐 | 资讯 | 视频
- 动态流（关注 Tab）：关注的人/热门钓获/新二手商品
- **资讯流（资讯 Tab）**：聚合路亚相关行业新闻、赛事、新品发布
- **短视频（视频 Tab）**：钓鱼视频、技巧教学、渔获展示

**快捷入口：** 天气查询、潮汐、新手入门

**交互：**
- 下拉刷新
- 无限滚动
- 点击动态/资讯/视频查看详情

---

### 2. 钓点 📍

**功能：**
- 地图模式：显示附近钓点（热力图）
- 列表模式：按距离/热度/鱼种筛选
- 钓点详情：
  - 位置、鱼种、最佳季节
  - 用户评价、作钓记录
  - 导航按钮
- 添加钓点：用户可提交新钓点（审核后上线）

**筛选条件：**
- 距离（附近/同城/全国）
- 鱼种（鲈鱼、鳜鱼、翘嘴、马口等）
- 水域类型（水库/河流/湖泊/海钓）
- 收费情况（免费/收费）

---

### 3. 发布 📝

**发布类型：**
1. **作钓记录**
   - 钓点（可选已收录的）
   - 日期、天气、水温
   - 使用拟饵（品牌、型号、颜色）
   - 渔获（鱼种、尺寸、照片）
   - 心得描述

2. **二手商品**
   - 分类（拟饵/竿/轮/配件）
   - 品牌、型号、成色
   - 价格（原价/现价）
   - 照片（最多 9 张）
   - 描述

3. **技巧/问答**
   - 标题、内容、图片
   - 标签（#新手 #鲈鱼 #拟饵选择）

---

### 4. 市场 🛒

**功能：**
- 分类浏览：拟饵 / 竿子 / 轮子 / 配件 / 其他
- 筛选：价格区间、品牌、成色、地区
- 搜索：关键词搜索
- 商品详情：
  - 图片轮播
  - 商品信息、卖家信息
  - 私信按钮、收藏按钮
- 发布入口：悬浮按钮

**交易流程：**
```
发布商品 → 买家私信 → 协商价格 → 线下/线上交易 → 双方评价
```

**注意：** 平台仅提供信息撮合，不介入资金交易（避免资质问题）

---

### 5. 我的 👤

**功能：**
- 个人信息：头像、昵称、签名、地区
- 数据统计：
  - 作钓次数、渔获数量、最长大鱼
  - 发布动态数、二手成交数
- 我的内容：
  - 作钓记录
  - 二手商品（在售/已售）
  - 收藏的钓点/商品
  - 发布的技巧/问答
- 消息中心：
  - 私信
  - 系统通知（点赞、评论、收藏）
- 设置：
  - 隐私设置
  - 账号绑定
  - 反馈/帮助

---

### 6. 资讯聚合 📰（新增）

**内容来源：**
- 路亚网站/论坛（路亚之家、钓鱼人等）
- 品牌官方公众号/官网（Shimano、Daiwa、Megabass 等）
- 赛事资讯（路亚锦标赛、公开赛）
- 行业媒体（渔具新品发布、评测）

**抓取策略：**
- 定时爬虫（每天 2-4 次）
- RSS 订阅（如果支持）
- API 对接（部分平台开放接口）
- 用户投稿（UGC 补充）

**展示形式：**
- 列表：标题 + 摘要 + 来源 + 时间
- 详情：跳转原文或抓取全文（注意版权）
- 分类：新闻/赛事/评测/教学/新品

**合规注意：**
- 标注来源和原文链接
- 不存储全文（仅摘要 + 跳转）
- 遵守 robots.txt

---

### 7. 短视频 📹（新增）

**内容来源：**
- **运营上传**：官方精选内容、合作博主、赛事集锦
- **用户分享**：作钓记录时同步上传视频、单独发布视频动态

**展示形式：**
- 竖屏全屏滑动（类似抖音）
- 信息流卡片（类似朋友圈视频）
- 可切换模式

**功能：**
- 点赞、评论、分享
- 关注作者
- 收藏视频
- **关联商品**：视频里用的拟饵可标记，跳转二手市场或拟饵库
- **关联钓点**：视频拍摄地点可标记钓点

**上传规范：**
- 格式：MP4/MOV
- 时长：15 秒 - 5 分钟
- 大小：≤50MB
- 审核：先审后发（敏感内容过滤）

**运营后台：**
- 内容管理（上架/下架/推荐）
- 用户投稿审核
- 数据统计（播放量、互动率）

## 核心交互流程

### 作钓记录流程
```
点击发布 → 选择"作钓记录" → 
填写钓点/天气/拟饵/渔获 → 
上传照片 → 添加心得 → 发布
```

### 二手交易流程
```
买家：浏览市场 → 查看详情 → 私信卖家 → 协商 → 交易 → 评价
卖家：发布商品 → 回复咨询 → 确认交易 → 标记已售 → 评价买家
```

### 钓点查询流程
```
进入钓点页 → 地图/列表切换 → 
筛选条件 → 查看详情 → 
查看作钓记录 → 导航前往
```

## 数据模型（初步）

### 用户表 (users)
```
id, openid, nickname, avatar, region, 
created_at, credit_score
```

### 钓点表 (spots)
```
id, name, location, lat, lng, fish_types, 
water_type, fee_status, best_season, 
status (待审核/已上线), created_by
```

### 作钓记录表 (logs)
```
id, user_id, spot_id, date, weather, 
water_temp, lures_used, catches, 
photos, notes, created_at
```

### 二手商品表 (items)
```
id, user_id, category, brand, model, 
condition, price, original_price, 
description, photos, status (在售/已售), 
created_at, updated_at
```

### 消息表 (messages)
```
id, sender_id, receiver_id, content, 
type (text/image), is_read, created_at
```
