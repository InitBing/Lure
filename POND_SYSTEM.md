# LureBin 黑坑管理系统

## 功能概述

黑坑管理系统为路亚钓友提供一站式黑坑钓场服务，包括：

- **塘主入驻**: 钓场老板可认证入驻平台
- **钓场管理**: 创建和管理钓场信息
- **活动发布**: 发布正钓、偷驴、比赛等活动
- **在线报名**: 钓友可在线报名缴费
- **评价系统**: 钓友可对钓场进行评价

## 核心流程

### 1. 塘主入驻流程

```
申请认证 → 提交资料 → 平台审核 → 审核通过 → 开始经营
```

**所需资料：**
- 钓场名称
- 联系人信息
- 联系电话
- 营业执照 (可选)
- 钓场照片

### 2. 活动发布流程

```
选择钓场 → 填写活动信息 → 设置费用/名额 → 发布活动 → 钓友报名
```

**活动信息：**
- 活动类型 (正钓/偷驴/比赛)
- 活动日期时间
- 报名费用
- 押金金额
- 奖金池
- 最大人数
- 钓场规则

### 3. 钓友报名流程

```
浏览活动 → 选择报名 → 支付费用 → 报名成功 → 现场签到 → 参与作钓 → 提交评价
```

**报名状态：**
- 已报名 (待支付)
- 已支付 (报名成功)
- 已签到
- 已完成
- 已取消 (已退款)

## 数据库表结构

### pond_owners (塘主表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 关联用户 ID |
| business_name | VARCHAR | 钓场名称 |
| contact_name | VARCHAR | 联系人 |
| contact_phone | VARCHAR | 联系电话 |
| verify_status | TINYINT | 认证状态 |
| credit_score | INT | 信用分 |

### ponds (钓场表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| owner_id | BIGINT | 塘主 ID |
| name | VARCHAR | 钓场名称 |
| province/city | VARCHAR | 地理位置 |
| water_type | TINYINT | 水域类型 |
| total_area | DECIMAL | 总面积 (亩) |
| fish_types | JSON | 鱼种 |
| facilities | JSON | 设施 |

### pond_events (活动表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| pond_id | BIGINT | 钓场 ID |
| title | VARCHAR | 活动标题 |
| event_type | TINYINT | 类型 (1 正钓 2 偷驴 3 比赛) |
| event_date | DATE | 活动日期 |
| fee_amount | DECIMAL | 报名费 |
| prize_pool | DECIMAL | 总奖金 |
| max_participants | INT | 最大人数 |
| status | TINYINT | 状态 |

### event_participants (报名表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| event_id | BIGINT | 活动 ID |
| user_id | BIGINT | 用户 ID |
| fee_amount | DECIMAL | 报名费 |
| payment_status | TINYINT | 支付状态 |
| check_in_status | TINYINT | 签到状态 |
| catch_weight | DECIMAL | 渔获重量 |
| catch_rank | INT | 排名 |

## API 接口

### 塘主相关

```
POST   /api/v1/pond/owner/apply      # 申请成为塘主
GET    /api/v1/pond/owner/info       # 获取塘主信息
```

### 钓场相关

```
POST   /api/v1/pond/ponds            # 创建钓场
GET    /api/v1/pond/ponds            # 钓场列表
GET    /api/v1/pond/ponds/:id        # 钓场详情
```

### 活动相关

```
POST   /api/v1/pond/events           # 发布活动
GET    /api/v1/pond/events           # 活动列表
GET    /api/v1/pond/events/:id       # 活动详情
POST   /api/v1/pond/events/:id/join  # 报名活动
```

### 报名相关

```
GET    /api/v1/pond/participants     # 我的报名
POST   /api/v1/pond/participants/:id/pay  # 支付报名费
```

### 评价相关

```
POST   /api/v1/pond/ponds/:id/review # 钓场评价
```

## 活动类型说明

### 正钓 (1)

- 新鱼投放，鱼情较好
- 费用较高
- 通常周末举行
- 按放鱼量收费

### 偷驴 (2)

- 正钓后的场次
- 费用较低
- 考验技术
- 性价比高

### 比赛 (3)

- 竞技性质
- 有奖金/奖品
- 需要裁判
- 按重量或尾数排名

- 塘主入驻：**免费**

## 规则说明

### 塘主规则

1. 提供真实钓场信息
2. 按约定投放鱼量
3. 公平对待钓友
4. 及时处理投诉
5. 不得私下交易

### 钓友规则

1. 遵守钓场规定
2. 爱护环境
3. 服从裁判安排
4. 不得作弊
5. 文明钓鱼

### 违规处理

- 警告
- 扣除信用分
- 暂停活动发布
- 永久封禁

## 使用指南

### 塘主入驻

1. 进入黑坑页面
2. 点击"塘主入驻"
3. 填写认证信息
4. 提交等待审核
5. 审核通过后即可发布钓场和活动

### 发布活动

1. 登录塘主账号
2. 选择要发布活动的钓场
3. 点击"发布活动"
4. 填写活动信息
5. 设置费用和名额
6. 确认发布

### 钓友报名

1. 浏览黑坑列表
2. 选择感兴趣的钓场
3. 查看近期活动
4. 点击"立即报名"
5. 支付报名费
6. 报名成功，等待活动开始

## 常见问题

**Q: 塘主认证需要多长时间？**
A: 通常 1-3 个工作日完成审核。

**Q: 报名后能退款吗？**
A: 活动开始前 24 小时可申请退款，之后不予退款。

**Q: 如何修改活动信息？**
A: 塘主可在活动管理页面修改未开始的活动信息。

**Q: 遇到纠纷怎么办？**
A: 可在平台发起投诉，管理员会介入处理。
