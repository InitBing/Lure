```
├─────────────────────────┤
│ [推荐 Banner 轮播]       │  ← 480x200
├─────────────────────────┤
│ 关注 | 推荐 | 资讯 | 视频│  ← Tab 切换
├─────────────────────────┤
│ ┌───────────────────┐   │
│ │ 👤 用户名         │   │
│ │ 📍 钓点名称       │   │
│ │ 🐟 鲈鱼 52cm      │   │
│ │ [图片 9:16]       │   │
│ │ ❤️ 23  💬 5  ⭐ 8  │   │
│ └───────────────────┘   │
│                         │
│ ┌───────────────────┐   │
│ │ 👤 用户名         │   │
│ │ 📍 钓点名称       │   │
│ │ 🐟 鳜鱼 38cm      │   │
│ │ [图片 9:16]       │   │
│ │ ❤️ 15  💬 2  ⭐ 3  │   │
│ └───────────────────┘   │
│                         │
│      ...滚动加载...      │
├─────────────────────────┤
│ 🏠   📍   ➕   🛒   👤  │
└─────────────────────────┘
```

### 交互说明
- 搜索栏：点击跳转搜索页
- Banner：轮播，点击跳转详情
- Tab：切换内容流
- 动态卡片：点击查看详情，可点赞/评论/收藏
- 下拉刷新，上拉加载更多
